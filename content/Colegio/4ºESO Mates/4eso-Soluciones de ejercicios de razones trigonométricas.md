# Ejercicio 11
**Problema:** Verifica la identidad $\tan^2(x) + 1 = \sec^2(x)$ para $x = \frac{\pi}{6}$.

**Solución:**
1. Calculamos $\tan\left(\frac{\pi}{6}\right) = \frac{1}{\sqrt{3}}$.
2. Elevamos al cuadrado: $\tan^2\left(\frac{\pi}{6}\right) = \frac{1}{3}$.
3. Calculamos $\sec\left(\frac{\pi}{6}\right) = \frac{1}{\cos\left(\frac{\pi}{6}\right)} = \frac{2}{\sqrt{3}}$.
4. Elevamos al cuadrado: $\sec^2\left(\frac{\pi}{6}\right) = \frac{4}{3}$.
5. Verificamos la identidad: $\tan^2\left(\frac{\pi}{6}\right) + 1 = \frac{1}{3} + 1 = \frac{4}{3} = \sec^2\left(\frac{\pi}{6}\right)$.

**Respuesta:** La identidad se verifica.

# Ejercicio 12
**Problema:** Si $\cot(\phi) = \frac{7}{24}$ y $\phi$ está en el segundo cuadrante, encuentra $\sin(\phi)$ y $\cos(\phi)$.

**Solución:**
1. Sabemos que $\cot(\phi) = \frac{\cos(\phi)}{\sin(\phi)}$ y $\cot(\phi) = \frac{7}{24}$.
2. Usamos el triángulo rectángulo para encontrar $r = \sqrt{7^2 + 24^2} = 25$.
3. Calculamos:
   - $\sin(\phi) = \frac{24}{25}$ (positivo en el segundo cuadrante).
   - $\cos(\phi) = -\frac{7}{25}$ (negativo en el segundo cuadrante).

**Respuesta:** $\sin(\phi) = \frac{24}{25}$ y $\cos(\phi) = -\frac{7}{25}$.

# Ejercicio 13
**Problema:** Resuelve la ecuación trigonométrica $\sin^2(x) = \cos^2(x)$ para $x \in [0, 2\pi]$.

**Solución:**
1. Sabemos que $\sin^2(x) = \cos^2(x)$ implica $\tan^2(x) = 1$.
2. Esto significa que $\tan(x) = \pm 1$.
3. Los valores de $x$ en $[0, 2\pi]$ son:
   - $x = \frac{\pi}{4}, \frac{3\pi}{4}, \frac{5\pi}{4}, \frac{7\pi}{4}$.

**Respuesta:** $x = \frac{\pi}{4}, \frac{3\pi}{4}, \frac{5\pi}{4}, \frac{7\pi}{4}$.

# Ejercicio 14
**Problema:** Simplifica la expresión $\frac{\tan(x)}{\sec(x)}$.

**Solución:**
1. Expandimos $\tan(x)$ y $\sec(x)$: $\frac{\tan(x)}{\sec(x)} = \frac{\frac{\sin(x)}{\cos(x)}}{\frac{1}{\cos(x)}}$.
2. Simplificamos: $\frac{\tan(x)}{\sec(x)} = \sin(x)$.

**Respuesta:** $\sin(x)$.

# Ejercicio 15
**Problema:** Si $\tan(\alpha) = \frac{3}{4}$ y $\alpha$ está en el primer cuadrante, calcula $\sec(\alpha)$.

**Solución:**
1. Usamos el triángulo rectángulo: $\tan(\alpha) = \frac{3}{4} \implies r = \sqrt{3^2 + 4^2} = 5$.
2. Calculamos:
   - $\cos(\alpha) = \frac{4}{5}$.
   - $\sec(\alpha) = \frac{1}{\cos(\alpha)} = \frac{5}{4}$.

**Respuesta:** $\sec(\alpha) = \frac{5}{4}$.

# Ejercicio 16
**Problema:** Prueba que $\sec^2(x) - \tan^2(x) = 1$ para $x = \frac{\pi}{3}$.

**Solución:**
1. Calculamos $\tan\left(\frac{\pi}{3}\right) = \sqrt{3}$.
2. Elevamos al cuadrado: $\tan^2\left(\frac{\pi}{3}\right) = 3$.
3. Calculamos $\sec\left(\frac{\pi}{3}\right) = \frac{1}{\cos\left(\frac{\pi}{3}\right)} = 2$.
4. Elevamos al cuadrado: $\sec^2\left(\frac{\pi}{3}\right) = 4$.
5. Verificamos la identidad: $\sec^2\left(\frac{\pi}{3}\right) - \tan^2\left(\frac{\pi}{3}\right) = 4 - 3 = 1$.

**Respuesta:** La identidad se verifica.

# Ejercicio 17
**Problema:** Resuelve para $x$: $\tan(x) = -1$ con $x \in [0, 2\pi]$.

**Solución:**
1. Sabemos que $\tan(x) = -1$ ocurre cuando $\tan(x)$ es negativo, es decir, en el segundo y cuarto cuadrante.
2. Los valores son:
   - $x = \pi - \frac{\pi}{4} = \frac{3\pi}{4}$.
   - $x = 2\pi - \frac{\pi}{4} = \frac{7\pi}{4}$.

**Respuesta:** $x = \frac{3\pi}{4}, \frac{7\pi}{4}$.

# Ejercicio 18
**Problema:** Si $\sin(\theta) = \frac{5}{13}$ y $\theta$ está en el segundo cuadrante, calcula todas las demás razones trigonométricas.

**Solución:**
1. Sabemos que $\sin(\theta) = \frac{5}{13}$ y $\cos^2(\theta) + \sin^2(\theta) = 1$.
2. Calculamos:
   - $\cos^2(\theta) = 1 - \sin^2(\theta) = 1 - \left(\frac{5}{13}\right)^2 = \frac{144}{169}$.
   - $\cos(\theta) = -\frac{12}{13}$ (negativo en el segundo cuadrante).
3. Calculamos las demás razones:
   - $\tan(\theta) = \frac{\sin(\theta)}{\cos(\theta)} = \frac{\frac{5}{13}}{-\frac{12}{13}} = -\frac{5}{12}$.
   - $\cot(\theta) = \frac{\cos(\theta)}{\sin(\theta)} = \frac{-\frac{12}{13}}{\frac{5}{13}} = -\frac{12}{5}$.
   - $\sec(\theta) = \frac{1}{\cos(\theta)} = \frac{13}{-12} = -\frac{13}{12}$.
   - $\csc(\theta) = \frac{1}{\sin(\theta)} = \frac{13}{5}$.

**Respuesta:** 
- $\sin(\theta) = \frac{5}{13}$.
- $\cos(\theta) = -\frac{12}{13}$.
- $\tan(\theta) = -\frac{5}{12}$.
- $\cot(\theta) = -\frac{12}{5}$.
- $\sec(\theta) = -\frac{13}{12}$.
- $\csc(\theta) = \frac{13}{5}$.

# Ejercicio 19
**Problema:** Demuestra que $\sin(2x) = 2\sin(x)\cos(x)$ para $x = \frac{\pi}{4}$.

**Solución:**
1. Calculamos $\sin\left(2 \cdot \frac{\pi}{4}\right) = \sin\left(\frac{\pi}{2}\right) = 1$.
2. Calculamos $2\sin\left(\frac{\pi}{4}\right)\cos\left(\frac{\pi}{4}\right)$:
   - $\sin\left(\frac{\pi}{4}\right) = \cos\left(\frac{\pi}{4}\right) = \frac{\sqrt{2}}{2}$.
   - $2\sin\left(\frac{\pi}{4}\right)\cos\left(\frac{\pi}{4}\right) = 2 \cdot \frac{\sqrt{2}}{2} \cdot \frac{\sqrt{2}}{2} = 1$.
3. Verificamos que $\sin(2x) = 2\sin(x)\cos(x)$.

**Respuesta:** La identidad se verifica.

# Ejercicio 20
**Problema:** Si $\cos(x) = -\frac{3}{5}$ y $x$ está en el tercer cuadrante, encuentra $\sin(x)$ y $\tan(x)$.

**Solución:**
1. Sabemos que $\cos(x) = -\frac{3}{5}$ y $\cos^2(x) + \sin^2(x) = 1$.
2. Calculamos:
   - $\sin^2(x) = 1 - \cos^2(x) = 1 - \left(-\frac{3}{5}\right)^2 = 1 - \frac{9}{25} = \frac{16}{25}$.
   - $\sin(x) = -\frac{4}{5}$ (negativo en el tercer cuadrante).
3. Calculamos $\tan(x) = \frac{\sin(x)}{\cos(x)} = \frac{-\frac{4}{5}}{-\frac{3}{5}} = \frac{4}{3}$.

**Respuesta:** $\sin(x) = -\frac{4}{5}$ y $\tan(x) = \frac{4}{3}$.
