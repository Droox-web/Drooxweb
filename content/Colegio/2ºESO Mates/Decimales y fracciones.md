### Representación Decimal y Fraccionaria para Números Racionales

1. **Relación entre fracción y decimal:**

   Un número racional es aquel que puede expresarse como el cociente de dos enteros, es decir, $\frac{a}{b}$, donde $a$ y $b$ son enteros y $b \neq 0$.

   - Para convertir una fracción a decimal, se realiza la división entre el numerador $a$ y el denominador $b$.
     - Ejemplo: $\frac{3}{4} = 3 \div 4 = 0.75$

   - Los decimales obtenidos pueden ser:
     - **Finitos**: Terminan después de un número fijo de dígitos.
       - Ejemplo: $\frac{1}{2} = 0.5$
     - **Infinitos periódicos**: Tienen un patrón repetitivo.
       - Ejemplo: $\frac{1}{3} = 0.\overline{3}$ (el $3$ se repite infinitamente).

2. **Conversión inversa (decimal a fracción):**

   - Si el decimal es **finito**, lo escribimos como una fracción con denominador $10^n$, donde $n$ es la cantidad de cifras decimales.
     - Ejemplo: $0.75 = \frac{75}{100} = \frac{3}{4}$.

   - Si el decimal es **periódico**, utilizamos un sistema algebraico para transformarlo:
     - Ejemplo: $x = 0.\overline{3}$. Multiplicamos por $10$ para desplazar el patrón:
       $$
       10x = 3.\overline{3}
       $$
       Restamos $x$:
       $$
       10x - x = 3.\overline{3} - 0.\overline{3} \implies 9x = 3
       $$
       Resolviendo: $x = \frac{1}{3}$.

### Relación con los Porcentajes

Un porcentaje es otra forma de representar un número racional, expresándolo como una fracción con denominador $100$.

1. Para convertir un decimal a porcentaje, se multiplica por $100$ y se añade el símbolo $\%$.
   - Ejemplo: $0.75 = 75\%$

2. Para convertir una fracción a porcentaje, se divide el numerador por el denominador, y el resultado se multiplica por $100$.
   - Ejemplo: $\frac{3}{4} = 0.75 \implies 75\%$

3. Inversamente, para convertir un porcentaje a fracción o decimal:
   - Como fracción: Dividimos entre $100$ y simplificamos.
     - Ejemplo: $75\% = \frac{75}{100} = \frac{3}{4}$
   - Como decimal: Dividimos entre $100$.
     - Ejemplo: $75\% = 0.75$
