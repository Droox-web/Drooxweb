1. $$ \frac{2x}{x + 3} + \frac{3}{x + 3} = \frac{2x + 3}{x + 3} $$

---

2. $$ \frac{4x}{x - 2} + \frac{5}{x + 2} = \frac{4x(x + 2) + 5(x - 2)}{(x - 2)(x + 2)} = \frac{4x^2 + 8x + 5x - 10}{x^2 - 4} = \frac{4x^2 + 13x - 10}{x^2 - 4} $$

---

3. $$ \frac{3x + 2}{x^2 - 1} + \frac{x - 1}{x^2 - 1} = \frac{(3x + 2) + (x - 1)}{x^2 - 1} = \frac{4x + 1}{x^2 - 1} $$

---

4. $$ \frac{x}{x + 4} + \frac{2x + 8}{x^2 - 16} = \frac{x(x - 4) + (2x + 8)}{(x + 4)(x - 4)} = \frac{x^2 - 4x + 2x + 8}{x^2 - 16} = \frac{x^2 - 2x + 8}{x^2 - 16} $$

---

5. $$ \frac{2x + 1}{x - 3} + \frac{4x}{x + 3} = \frac{(2x + 1)(x + 3) + 4x(x - 3)}{(x - 3)(x + 3)} = \frac{2x^2 + 6x + x + 3 + 4x^2 - 12x}{x^2 - 9} = \frac{6x^2 - 5x + 3}{x^2 - 9} $$

---

6. $$ \frac{5x}{x^2 - 9} + \frac{2}{x - 3} = \frac{5x + 2(x + 3)}{x^2 - 9} = \frac{5x + 2x + 6}{x^2 - 9} = \frac{7x + 6}{x^2 - 9} $$

---

7. $$ \frac{x^2 + 3x}{x + 1} + \frac{2x + 1}{x - 1} = \frac{(x^2 + 3x)(x - 1) + (2x + 1)(x + 1)}{(x + 1)(x - 1)} $$

    $$ = \frac{x^3 - x^2 + 3x^2 - 3x + 2x^2 + 3x + 1}{x^2 - 1} = \frac{x^3 + 4x^2 - x + 1}{x^2 - 1} $$

---

8. $$ \frac{3x + 4}{x^2 - 4} + \frac{x - 2}{x + 2} = \frac{(3x + 4) + (x - 2)(x - 2)}{x^2 - 4} = \frac{3x + 4 + x^2 - 4x + 4}{x^2 - 4} $$

    $$ = \frac{x^2 - x + 8}{x^2 - 4} $$

---

9. $$ \frac{2x + 3}{x - 5} + \frac{x^2 - 25}{x + 5} = \frac{(2x + 3)(x + 5) + (x^2 - 25)(x - 5)}{(x - 5)(x + 5)} $$

    $$ = \frac{2x^2 + 10x + 3x + 15 + x^3 - 25x - 5x^2 + 125}{x^2 - 25} = \frac{x^3 - 3x^2 - 12x + 140}{x^2 - 25} $$

---

10. $$ \frac{4x}{x^2 - 16} + \frac{6}{x + 4} = \frac{4x + 6(x - 4)}{x^2 - 16} = \frac{4x + 6x - 24}{x^2 - 16} = \frac{10x - 24}{x^2 - 16} $$
