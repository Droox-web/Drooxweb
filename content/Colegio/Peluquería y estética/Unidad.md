Una unidad es una medida estándar que se utiliza para cuantificar y expresar cantidades de diferentes [[Magnitud|magnitudes]]. 

Por ejemplo, para medir longitud, el metro es una unidad; para medir tiempo, el segundo es otra unidad.

Las unidades permiten expresar y comparar valores de forma consistente y comprensible. 

Se organizan en sistemas de unidades, como el [[Sistema Internacional]] de Unidades (SI), que define unidades básicas y derivadas para diversas magnitudes físicas.
