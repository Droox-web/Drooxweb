El Sistema Internacional de Unidades (SI) es el sistema de medida más ampliamente aceptado en el mundo para expresar y cuantificar magnitudes físicas. 

Fue creado en 1960 por la Conferencia General de Pesas y Medidas con el objetivo de estandarizar las unidades de medida, permitiendo que la ciencia, la industria y el comercio tengan un marco común para medir y comunicar de manera precisa y consistente.

El SI se basa en **siete unidades fundamentales**, cada una correspondiente a una magnitud física básica:

1. **Metro (m)** para la longitud
2. **Kilogramo (kg)** para la masa
3. **Segundo (s)** para el tiempo
4. **Amperio (A)** para la corriente eléctrica
5. **Kelvin (K)** para la temperatura
6. **Mol (mol)** para la cantidad de sustancia
7. **Candela (cd)** para la intensidad luminosa

A partir de estas unidades fundamentales, se pueden construir unidades **derivadas** para magnitudes como el área, el volumen, la velocidad, la fuerza, entre muchas otras. Por ejemplo, la unidad de velocidad es el metro por segundo (m/s), y la unidad de fuerza es el newton (N), que se define como kg·m/s².

El SI es revisado y actualizado periódicamente para reflejar avances científicos y mantener la precisión en las mediciones, como la redefinición del kilogramo en 2019, que se basó en constantes físicas fundamentales en lugar de en un objeto físico.
